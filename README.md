To set up the project, users need to run the following commands


# 1. Install Laravel Socialite
composer require laravel/socialite   
(After that make your Setup Google App | OAuth 2.0 client);

# 2. Instalează Laravel UI
composer require laravel/ui

# 3. Instalează dependențele pentru dezvoltare
npm install

# 4. Instalează Vite și plugin-ul Vue pentru Vite
npm install --save-dev @vitejs/plugin-vue

# 5. Instalează Axios
npm install axios

# 6. Install Echo Server
npm install -g laravel-echo-server

# 7. Install 
laravel-echo-server init

# 8. Install 
composer require laravel/ui
